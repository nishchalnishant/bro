# NEET PG Preparation Repository — Master Index

## Repository Overview

This repository is a structured, exam-focused knowledge base for NEET PG (National Eligibility cum Entrance Test — Postgraduate) preparation. It contains subject notes, PYQ analysis, quick revision materials, comparison tables, mnemonics, and study strategy resources, organized to support preparation from first reading through final revision.

All content is optimized for the current NEET PG format: **200 MCQs / 800 marks / 3.5 hours / +4 per correct / −1 per incorrect**.

---

## Exam Pattern Summary

| Parameter | Details |
|---|---|
| Full name | NEET PG (conducted by NBE/NMC) |
| Total questions | 200 MCQs |
| Total marks | 800 |
| Duration | 3 hours 30 minutes |
| Marking scheme | +4 (correct), −1 (incorrect), 0 (unanswered) |
| Mode | Computer-based test (CBT) |
| Language | English |
| Frequency | Annually (typically March–April) |
| Conducted by | National Board of Examinations (NBE) / NMC |
| Eligibility | MBBS degree with internship completion |
| Score validity | 1 year from date of result |

**Cutoff scores (approximate, varies by category):**
- General: ~275–300+ (varies by year and competition)
- OBC/SC/ST: Lower cutoffs as per reservation policy

## How to Use This Repository

### The Two-File System

Every subject folder contains exactly two files:

| File | Purpose | When to Use |
|---|---|---|
| `*-lecture.md` | Detailed first-principles lecture notes — mechanisms, pathophysiology, full explanations | Phase 1 (first reading, months 1–6): read top-to-bottom to build deep understanding |
| `*-notes.md` | Condensed exam-focused rapid revision notes — high-yield facts, tables, one-liners, key numbers | Phase 3–4 (revision, months 10–12): rapid cycling through core facts before the exam |

**Rule of thumb:** Open the lecture file when you are learning. Open the notes file when you are revising.

## Study Roadmap — Phase-Wise

### Phase 1: Foundation and Reading (Months 1–8)
**Goal:** Complete one solid read of all 19 subjects.
- Choose one primary textbook per subject (see recommended-books.md)
- Annotate textbooks; do not make elaborate separate notes
- Solve chapter-end PYQs immediately after each chapter
- Build one-liner notebook from MCQ explanations

**Progress milestones:**
- Month 2: Anatomy + Physiology + Biochemistry complete
- Month 4: Pathology + Pharmacology complete
- Month 5: Microbiology + Forensic + SPM complete
- Month 7: Medicine + Surgery + OBG complete
- Month 8: Pediatrics + ENT + Ophthalmology + Psychiatry + Derm + Ortho + Anesthesia + Radiology complete

### Phase 2: MCQ Phase (Months 4–10, Overlapping)
**Goal:** Subject-wise MCQ mastery.
- Start subject QBank immediately after finishing each subject
- Target: 50–100 MCQs/day per subject being practiced
- Mandatory error analysis for every wrong answer

### Phase 3: Revision Phase (Months 9–11)
**Goal:** Consolidate, revise, plug gaps.
- Year-wise NEET PG papers (start with 2024, work backward)
- Rotate through one-liners, mnemonics, comparison tables
- Re-read only annotated/starred content in textbooks
- One complete subject revision every 10–14 days

### Phase 4: Grand Test Phase (Months 10–12)
**Goal:** Performance optimization and exam simulation.
- Frequency: 1/week → 2/week → 3–4/week in final month
- Full 3.5-hour simulated exam conditions
- Post-test analysis: subject-wise accuracy, category analysis, error notebook update

## Study Tips

**On maintaining pace:**
- The preparation marathon is 12 months. The first 2 months are the hardest — after that, momentum builds.
- Do not compare daily page counts with others. Depth of understanding matters more than speed.
- Revision matters more than first reading. Plan for 3–4 complete revisions, not just 1.

**On managing multiple subjects:**
- After finishing a subject, do not abandon it. Set a 10–15 minute weekly recall slot for subjects already covered.
- The brain forgets — systematic spaced repetition (via PYQs + one-liners) maintains retention across all subjects simultaneously.

**On the marking scheme:**
- −1 for wrong answers means do not guess blindly. If you can eliminate 2 options, calculated guessing is favorable.
- At 50% elimination (2 wrong options identified), probability of right answer = 50%; expected marks = 0.5×4 − 0.5×1 = +1.5 (worth attempting).
- If zero elimination is possible, do not attempt — expected marks = 4×(1/4) − 1×(3/4) = 1 − 0.75 = +0.25 (marginally positive, but risk of mental accounting distorts performance).

**On high-yield focus:**
- The top 5 subjects (Medicine, Pathology, Pharmacology, SPM, Anatomy) contribute ~85+ questions. Mastery here shifts the score significantly.
- Even in "small" subjects (Anesthesia, Radiology), the same 4–5 core topics appear every year. Focused preparation yields disproportionate returns.

**On stress management:**
- High-stress periods (mock test results, self-assessment) are data, not verdicts. Poor mock performance in month 6 is an early warning that month 11 can fix.
- Consistent 10–12 hours of focused work beats sporadic 15-hour sessions followed by burnout.
- Sleep and exercise are non-negotiable for memory consolidation. Minimum 6–7 hours of sleep.
