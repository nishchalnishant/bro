# NEET PG — 12-Month Study Schedule

## Exam at a Glance

| Parameter | Value |
|---|---|
| Total questions | 200 MCQs |
| Total marks | 800 |
| Duration | 3 hours 30 minutes |
| Correct answer | +4 marks |
| Incorrect answer | −1 mark |
| Unanswered | 0 marks |
| Subjects | 19 |

### Subject Tier / Weightage

| Tier | Subjects | Approx. Questions |
|---|---|---|
| Tier 1 (High-yield) | General Medicine, Pathology, Pharmacology, SPM, Anatomy | 85–95 |
| Tier 2 | General Surgery, OBG, Physiology, Microbiology, Biochemistry, Pediatrics | 75–85 |
| Tier 3 | ENT, Ophthalmology, Dermatology, Orthopedics, Forensic Medicine, Psychiatry | 35–40 |
| Tier 4 | Anesthesia, Radiology | 8–9 |

**Strategy implication:** Perfect the Tier 1 subjects first. Tier 4 subjects can be covered in a single intensive week each.

## Daily Study Plan Template

| Time Block | Duration | Activity |
|---|---|---|
| Morning | 3–4 hours | New content — read lecture file (`*-lecture.md`) chapter by chapter |
| Afternoon | 2–3 hours | MCQ practice — subject-wise QBank or PYQ set |
| Evening | 1–2 hours | Revision of yesterday's material — open yesterday's notes file (`*-notes.md`) and scan key points |
| Night | 30 minutes | Error analysis — review wrong answers from afternoon session; add one-liners to personal notebook |

**Total study time:** 7–10 hours/day (ramp from 7 in Month 1 to 10 by Month 6; maintain 10–12 in revision phase).

**Daily MCQ targets:**

|---|---|
| Month 1–2 | 50/day (chapter-end Qs only) |
| Month 3–4 | 75–100/day |
| Month 5–6 | 100–150/day |
| Month 7–8 | 150/day |

## Grand Test Schedule and Analysis Protocol

### When to Take Grand Tests

| Period | Frequency |
|---|---|
| Month 9 | 1 per week |
| Month 10 | 2 per week |
| Month 11 | 2–3 per week |
| Month 12 (weeks 1–3) | 3–4 per week |
| Final 7 days | 1 confidence GT only |

### Post-Test Analysis Protocol (mandatory — do not skip)

1. **Immediate review (within 2 hours of test):** Go through every wrong answer. Do not just read the correct answer — read the full explanation.
2. **Error classification:** Tag each error as (a) knowledge gap, (b) silly mistake/misread, (c) guessing error.
3. **Error notebook entry:** Add all knowledge-gap errors to `07-Grand-Tests/Error-Notebook/error-notebook-template.md` with subject + topic + correct fact.
4. **Performance tracker update:** Log score, subject-wise accuracy, and total percentile in `07-Grand-Tests/Performance-Tracker/performance-tracker.md`.
5. **Next-day targeted revision:** Spend 1 hour on the top 3 error topics from the previous day's test.

### Performance Thresholds

| Grand Test Score | Interpretation | Action |
|---|---|---|
| > 65% accuracy | On track | Maintain pace, identify remaining weak spots |
| 50–65% accuracy | Acceptable (early phase) | Increase revision frequency for weak subjects |
| < 50% accuracy | Warning | Do not take more tests — spend 1 week doing targeted notes revision for weak subjects, then retest |
| Consistently < 40% in 1 subject | Critical | Restart that subject's notes file from scratch |

## Subject-Specific Strategy Notes

| Subject | Key Strategy |
|---|---|
| Anatomy | Focus on neuroanatomy, clinical anatomy (hernias, spaces), radiological anatomy |
| Physiology | CVS + Renal + Endocrine are maximum-yield; derive do not memorize |
| Biochemistry | Enzymes, vitamins, and inborn errors = repeat topics every year |
| Pathology | Cell injury + Hematology + Neoplasia = core; learn morphology with images |
| Pharmacology | Mechanism of action + adverse effects + drug interactions = exam currency |
| Microbiology | Culture media, stains, virulence factors, and vaccines are direct Q sources |
| Medicine | DM, thyroid, cardiology, nephrology — learn diagnostic criteria and management algorithms |
| Surgery | High-yield: obstructive jaundice, intestinal obstruction, hernias, thyroid surgery |
| OBG | Partograph, Bishop score, APH/PPH causes, contraceptive failure rates |
| Pediatrics | Immunization schedule (update annually), milestones, neonatology |
| SPM | Epidemiological formulas, national program statistics (update from MoHFW annually) |
| ENT | Audiogram patterns, tympanogram, vertigo differential, laryngeal anatomy |
| Ophthalmology | Glaucoma types + drugs, retinal disorders, cataracts, pupil reactions |
| Dermatology | Drug reactions, STDs, psoriasis/eczema differentiation |
| Forensic | Thanatology sequence, poison antidotes, medico-legal sections of IPC/CrPC |
| Orthopedics | Fracture complications, X-ray findings, compartment syndrome |
| Psychiatry | ICD vs DSM criteria, drug choices, rating scales |
| Anesthesia | MAC values, reversal agents, critical events management |
| Radiology | X-ray signs, CT findings — pattern recognition is key |
