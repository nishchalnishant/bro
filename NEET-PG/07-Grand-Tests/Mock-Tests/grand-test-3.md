**Correct: B) Acute ischemic stroke (or pyogenic abscess/epidermoid cyst) — true restricted diffusion: DWI bright + ADC dark = failure of water molecule diffusion due to cytotoxic edema** — True restricted diffusion = DWI hyperintense + ADC hypointense; caused by: acute infarct (cytotoxic edema), pyogenic abscess (viscous pus), epidermoid cyst, hypercellular tumors; T2 shine-through (vasogenic edema) = DWI bright + ADC bright — not true restriction.
