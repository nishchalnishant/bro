## Cell Injury

### Reversible vs Irreversible Injury

> **PYQ HOTSPOT**: Nuclear changes in irreversible injury, cell swelling mechanism

**Reversible Injury** (cell can recover):
- **Cellular swelling** (most common, earliest change): Na+/K+ ATPase fails → Na+ and H2O enter
- **Fatty change (steatosis)**: Fat accumulates; liver most common; reversible
- Mitochondrial swelling, ER dilation, ribosome detachment
- **Plasma membrane blebs** (not ruptured)

**Irreversible Injury** (cell death):
- Plasma membrane disruption
- **Mitochondrial permeability transition pore (mPTP)** opening
- Nuclear changes (key hallmarks):

| Nuclear Change | Description | Cause |
|---------------|-------------|-------|
| **Pyknosis** | Nuclear shrinkage, increased basophilia | First change |
| **Karyorrhexis** | Nuclear fragmentation | Follows pyknosis |
| **Karyolysis** | Nuclear dissolution/fading (pale) | DNases digest DNA |

**Mnemonic**: **"PKL"** = Pyknosis → Karyorrhexis → Karyolysis (sequence of nuclear death)

**Key mechanisms of cell death**:
1. ATP depletion → Na+/K+ pump failure → swelling
2. Ca2+ influx → activates phospholipases, proteases, endonucleases
3. Reactive oxygen species (ROS) → membrane damage
4. Loss of membrane integrity → cytoplasmic contents leak

### Apoptosis

> **PYQ HOTSPOT**: Intrinsic vs extrinsic pathway, Bcl-2, caspases, features

**Apoptosis vs Necrosis**:
| Feature | Apoptosis | Necrosis |
|---------|----------|---------|
| Mechanism | Programmed, energy-requiring | Passive, accidental |
| Cell size | **Shrinks** | Swells |
| Nucleus | Pyknosis → karyorrhexis | Karyolysis |
| Plasma membrane | Intact | Disrupted |
| Inflammation | **No** (anti-inflammatory) | **Yes** |
| DNA fragmentation | Ladder pattern (internucleosomal) | Random |
| **Apoptotic bodies** | Yes | No |

**Intrinsic Pathway (Mitochondrial)**:
- Triggers: DNA damage, oxidative stress, ischemia, p53 activation
- **Bcl-2 family**:
  - **Pro-apoptotic**: BAX, BAK, BID, BAD → form pores in mitochondria
  - **Anti-apoptotic (survival)**: **Bcl-2, Bcl-XL** → inhibit BAX/BAK
- → Cytochrome C release → Apoptosome → **Caspase 9** activation → executioner caspases (3, 6, 7)

**Extrinsic Pathway (Death Receptor)**:
- Triggers: FasL-FasR, TNF-TNFR1
- → DISC (Death-inducing signaling complex) → **Caspase 8** activation → executioner caspases

**Executioner caspases** (3, 6, 7): Cleave cytoskeletal proteins, CAD (endonuclease) → DNA laddering

**IAP (Inhibitor of Apoptosis Proteins)**: Block caspases (overexpressed in cancer)
**Bcl-2 overexpression**: **Follicular lymphoma** (t(14;18) — Bcl-2 overexpressed → no apoptosis)

### Necrosis Types

> **PYQ HOTSPOT**: Type by organ/disease - CRITICAL TABLE

| Necrosis Type | Mechanism | Appearance | Seen In |
|--------------|-----------|-----------|--------|
| **Coagulative** | Ischemia; proteins denatured but architecture preserved | Firm, pale; **ghost outlines of cells** | **All organs EXCEPT brain**; MI, renal infarct |
| **Liquefactive** | Enzymatic digestion (proteases); architecture lost | Liquid, creamy, pus | **Brain infarct** (also bacterial/fungal infections → pus) |
| **Caseous** | Combination; structureless; "cheese-like" | Soft, white, cottage cheese | **TB, fungi (Histoplasma)**; granulomatous disease |
| **Fat necrosis** | Lipases release FAs → Ca2+ soap formation | Chalky white deposits | **Pancreatitis, breast trauma** |
| **Fibrinoid** | Immune complex deposition + fibrin leak | Pink, homogeneous, glassy | **Vasculitis, autoimmune disease, malignant HTN** |
| **Gangrenous** | Superimposed infection (wet) or ischemia (dry) | Dry/Wet/Gas | Extremities, bowel |

## Inflammation

### Acute Inflammation

> **PYQ HOTSPOT**: Vascular events, exudate composition, mediators - HIGHLY TESTED

**Vascular Events** (in order):
1. **Transient vasoconstriction** (seconds)
2. **Vasodilation** (histamine, NO) → increased blood flow → redness, heat
3. **Increased vascular permeability** → exudate formation → swelling
4. **Stasis** → margination → rolling → adhesion → transmigration

**Cardinal Signs**: Rubor (redness), Calor (heat), Tumor (swelling), Dolor (pain), **Functio laesa** (loss of function)

**Cellular Events** (neutrophil recruitment):

| Step | Molecules |
|------|----------|
| **Margination** | Increased viscosity; WBC marginate |
| **Rolling** | **Selectins** (P-selectin on platelets/endothelium, E-selectin on endothelium, L-selectin on WBC) |
| **Adhesion (firm)** | **Integrins** (LFA-1, MAC-1 on WBC) + **ICAM-1, VCAM-1** on endothelium |
| **Transmigration** | **PECAM-1 (CD31)** at intercellular junctions |
| **Chemotaxis** | **C5a, IL-8 (CXCL8), LTB4, fMLP (bacterial)** |

**LAD (Leukocyte Adhesion Deficiency)**:
- Defect: CD18 (beta-2 integrin) → **no firm adhesion**
- Features: **Recurrent bacterial infections, no pus formation, delayed umbilical cord separation, neutrophilia**

### Inflammatory Mediators

> **PYQ HOTSPOT**: Source and action of each mediator

| Mediator | Source | Action |
|---------|--------|--------|
| **Histamine** | Mast cells, basophils, platelets | **First mediator released**; vasodilation, ↑ permeability; H1 receptor |
| **Serotonin** | Mast cells (rodents), platelets | Vasoconstriction, ↑ permeability |
| **PGE2, PGI2** | Arachidonic acid via COX | Vasodilation, pain sensitization, **fever** |
| **TXA2** | Platelets via COX | Vasoconstriction, platelet aggregation |
| **LTB4** | Arachidonic acid via 5-LOX | **Potent neutrophil chemotaxis** |
| **LTC4, LTD4, LTE4** | Mast cells, eosinophils via 5-LOX | Bronchoconstriction, ↑ permeability (**slow-reacting substances** in asthma) |
| **PAF (Platelet activating factor)** | Mast cells, basophils, endothelium | Vasodilation, ↑ permeability, platelet aggregation |
| **IL-1, TNF-α** | Macrophages | Acute phase reactants, fever, systemic effects; **endogenous pyrogens** |
| **IL-8 (CXCL8)** | Macrophages, endothelium | **Potent neutrophil chemotaxis** |
| **IL-6** | Macrophages, T cells | Acute phase response, **main inducer of CRP, fibrinogen** |
| **C3a, C5a** | Complement | Anaphylatoxins; **C5a = potent chemotaxis** |
| **C3b** | Complement | Opsonization |
| **NO (Nitric oxide)** | Endothelium (eNOS), macrophages (iNOS) | Vasodilation, kills microbes |
| **Bradykinin** | Plasma kininogens | **Pain**, vasodilation, ↑ permeability |

**Arachidonic Acid Pathway**:
- **COX pathway**: PGs, TXA2 (inhibited by NSAIDs, aspirin)
- **LOX pathway**: Leukotrienes (not inhibited by NSAIDs)
- Aspirin: **Irreversibly** inhibits COX (acetylates); unique platelet effect (no protein synthesis)

### Chronic Inflammation

**Features**: Macrophages, lymphocytes, plasma cells; tissue destruction + repair ongoing
**Main cell**: **Macrophage** (also called histiocyte in tissues)
- Activated macrophages: Epithelioid cells (in granulomas)
- Giant cells: Fusion of macrophages

**Granuloma Formation**:
- Definition: Focal collection of **activated macrophages (epithelioid cells)** ± multinucleated giant cells, surrounded by lymphocytes
- Trigger: T-cell (Th1) mediated; IL-12, IFN-γ crucial

| Granuloma Type | Necrosis | Giant Cell Type | Seen In |
|--------------|---------|----------------|--------|
| **Caseating** | **Yes (caseous)** | Langhans (peripheral nuclei) | **TB**, deep fungi |
| **Non-caseating** | No | Langhans or foreign body | **Sarcoidosis**, Crohn's, leprosy (tuberculoid), berylliosis |
| Foreign body | No | Foreign body type (central nuclei) | Foreign material |

### Wound Healing

> **PYQ HOTSPOT**: Primary vs secondary intention, factors, keloid vs hypertrophic scar

**Phases**:
1. **Hemostasis** (immediate): Platelet plug → fibrin clot
2. **Inflammation** (0-3 days): Neutrophils → macrophages (day 2-3)
3. **Proliferation** (day 3 - 3 weeks): Granulation tissue (fibroblasts + new vessels)
4. **Remodeling** (weeks-years): Collagen type III → type I; max strength ~80% original

**Primary vs Secondary Intention**:
| Feature | Primary (1°) | Secondary (2°) |
|---------|------------|---------------|
| Wound edges | Opposed (sutured) | Not opposed (open) |
| Scar | Minimal | Larger scar |
| Contraction | Minimal | **Significant** (myofibroblasts) |
| Granulation tissue | Less | More |

**Factors Impairing Healing**:
- **Local**: Infection, foreign body, ischemia, excessive tension
- **Systemic**: **Diabetes**, malnutrition (Vit C, zinc, protein), immunosuppression, corticosteroids, radiation

**Keloid vs Hypertrophic Scar**:
| Feature | Hypertrophic Scar | **Keloid** |
|---------|-----------------|----------|
| Extends beyond wound | No | **Yes** |
| Recurrence after excision | Rare | **Common** |
| Prone populations | Anyone | **African Americans** |
| Histology | Normal collagen bundles | Thick **whorled collagen** bundles |

## Neoplasia

### Carcinogenesis

> **PYQ HOTSPOT**: Steps, oncogenes, tumor suppressor genes

**Steps**:
1. **Initiation**: Irreversible DNA damage (mutagen); creates "initiated" cell
2. **Promotion**: Non-mutagenic; reversible; promotes initiated cells to proliferate
3. **Progression**: Additional mutations; invasive/malignant phenotype

**Key concepts**:
- **Carcinogen activation**: Many require metabolic activation (Phase I enzymes, CYP450)
- **Aflatoxin B1**: Liver carcinogen; **HCC** (G→T mutation in p53 codon 249)
- **Two-hit hypothesis (Knudson)**: Both alleles of tumor suppressor must be inactivated

### Benign vs Malignant Features

| Feature | Benign | Malignant |
|---------|--------|----------|
| Growth rate | Slow | Rapid |
| Borders | Well-defined, encapsulated | Poorly defined, infiltrating |
| Differentiation | Well-differentiated | Poorly differentiated (anaplastic) |
| Metastasis | **No** | **Yes** |
| Mitoses | Rare, normal | **Frequent, abnormal** |
| Nuclear:Cytoplasmic ratio | Normal | **Increased** |
| Necrosis | Rare | Common |

**Metastasis routes**:
- **Lymphatic**: Carcinomas (most common route for carcinomas)
- **Hematogenous**: Sarcomas (most common), also colorectal (portal → liver), prostate/breast/lung → bone
- **Seeding (transcoelomic)**: Ovarian cancer → peritoneal cavity (Krukenberg tumor)
- **CSF**: CNS tumors
- **Direct extension**: Locally invasive

**Sentinel lymph node**: First node draining a tumor; biopsy to stage (breast, melanoma)

## Hematology

### Anemia Classification

> **PYQ HOTSPOT**: MCV-based classification, lab findings

| Type | MCV | RDW | Causes |
|------|-----|-----|--------|
| **Microcytic** (<80 fL) | Low | High (IDA), Normal (thalassemia) | **IDA, Thalassemia, Sideroblastic, ACD** |
| **Normocytic** (80-100 fL) | Normal | Variable | **Acute blood loss, hemolysis, aplastic anemia, ACD** |
| **Macrocytic** (>100 fL) | High | Usually high | **B12/Folate deficiency**, liver disease, hypothyroidism, drugs (MTX, hydroxyurea) |

**Mnemonic for microcytic anemia**: **"TAILS"** = Thalassemia, ACD, IDA, Lead poisoning, Sideroblastic

---

### Iron Deficiency Anemia

> **PYQ HOTSPOT**: Stages, lab findings, peripheral smear

**Stages**:
1. **Pre-latent**: Depleted iron stores (↓ferritin); no anemia
2. **Latent (iron-depleted)**: ↓Serum iron, ↑TIBC; still no anemia
3. **Iron-deficiency anemia**: ↓Hb, microcytic hypochromic anemia

**Lab Findings**:
| Lab | IDA | ACD | Sideroblastic |
|-----|-----|-----|-------------|
| Serum Iron | ↓↓ | ↓ | ↑/Normal |
| TIBC | **↑↑** | **↓** | Normal |
| Serum Ferritin | **↓↓** (most sensitive) | **↑** | ↑ |
| % Saturation | ↓ | ↓ | ↑ |
| Bone marrow iron | Absent | Present (reticuloendothelial) | Ringed sideroblasts |

**Peripheral smear in IDA**: Microcytic, hypochromic, **pencil cells**, target cells, **Koilonychia** (spoon nails)
**Pica**: Eating non-food items (ice = pagophagia is most specific for IDA)

---

### Megaloblastic Anemia

**Peripheral smear**: **Macrocytes, hypersegmented neutrophils** (>5 lobes), oval macrocytes
**Bone marrow**: Megaloblasts (large cells, nuclear-cytoplasmic asynchrony)

**Cause**: Defective DNA synthesis (B12 or Folate deficiency); RNA/protein synthesis intact

*(See B12 vs Folate comparison in Biochemistry notes — same table repeated here for Pathology context)*

**Other causes of macrocytosis** (non-megaloblastic): Alcohol, liver disease, hypothyroidism, reticulocytosis, drugs

### Hemolytic Anemias

> **PYQ HOTSPOT**: Classification, lab findings, inheritance

**Lab findings in hemolysis (general)**:
- ↑ Unconjugated bilirubin, ↑ LDH, ↑ reticulocytes
- ↓ Haptoglobin (binds free Hb)
- Intravascular hemolysis: + Hemoglobinuria, hemoglobinemia, hemosiderinuria

| Disease | Defect | Inheritance | Peripheral Smear | Key Features |
|---------|--------|-------------|-----------------|-------------|
| **Hereditary Spherocytosis** | **Spectrin/Ankyrin/Band 3** | **AD** | **Spherocytes, no central pallor** | Splenomegaly; **MCH↑, MCHC↑**; aplastic crisis (Parvovirus B19); **osmotic fragility↑**; responds to splenectomy |
| **G6PD Deficiency** | **G6PD** | **X-linked recessive** | **Bite cells, Heinz bodies** | Triggered by oxidant stress; episodic hemolysis |
| **Sickle Cell Disease** | **Glu→Val (codon 6) HbS** | **AR** | **Sickle cells, target cells** | Sickling with hypoxia, acidosis, dehydration; vaso-occlusive crisis; **autosplenectomy**; **Howell-Jolly bodies** post-splenectomy |
| **Thalassemia** | **Globin chain synthesis defect** | AR | Target cells, microcytosis | See table below |
| **PNH** | **PIGA mutation** (GPI anchor) | Acquired | - | Episodic hemolysis with sleep (↓ pH); thrombosis (portal, hepatic veins); **Ham test / CD55, CD59 absent** |
| **Warm AIHA** | IgG auto-Ab | Acquired | Spherocytes | **DAT positive** (IgG); SLE, CLL, drugs (methyldopa) |
| **Cold AIHA** | IgM auto-Ab | Acquired | Agglutination | **Mycoplasma, EBV**; worse in cold |

**Thalassemia**:
| Type | Defect | Features |
|------|--------|---------|
| **Beta thalassemia major** | No β chain (β0/β0) | **Severe anemia** from 6 months; hepatosplenomegaly; **crew-cut skull XR**; **Cooley anemia**; transfusion-dependent |
| **Beta thalassemia minor** | ↓ β chain (β+/normal) | Mild microcytic; **target cells**; elevated HbA2 (>3.5%) |
| **Alpha thalassemia trait** | 2 deleted alpha genes | Mild microcytosis |
| **HbH disease** | 3 deleted alpha genes | Moderate hemolytic anemia; HbH = 4 beta chains |
| **Hydrops fetalis** | 4 deleted alpha genes | **Fatal in utero**; HbBart = 4 gamma chains |

**Sickle cell features**:
- **Vaso-occlusive crisis**: Bone pain, dactylitis (hand-foot syndrome in children), priapism
- **Aplastic crisis**: Parvovirus B19
- **Sequestration crisis**: Spleen traps sickle cells → rapid splenomegaly
- **Hemolytic crisis**: Rapid hemolysis
- **Long term**: Autosplenectomy → encapsulated organism infections (Salmonella osteomyelitis), Howell-Jolly bodies

### Aplastic Anemia

- **Pancytopenia** with **hypocellular bone marrow** (replaced by fat)
- Causes: Idiopathic (50%), drugs (chloramphenicol, sulfonamides), radiation, chemicals (benzene), viruses (EBV, HIV, hepatitis), **PNH**
- **Peripheral smear**: Normocytic normochromic anemia; no immature cells
- Treatment: **BMT** (first choice if <40 years, HLA-matched donor); Cyclosporine + ATG (others)

---

### Polycythemia Vera

- **JAK2 V617F mutation** (95%)
- Triad: **Erythrocytosis, leukocytosis, thrombocytosis**
- Features: Ruddy cyanosis, splenomegaly, **aquagenic pruritus** (after bath), **thrombosis** (Budd-Chiari)
- **Serum EPO**: **Low** (distinguishes from secondary polycythemia where EPO is ↑)
- Complications: Transformation to myelofibrosis or AML

### Hodgkin vs Non-Hodgkin Lymphoma

> **PYQ HOTSPOT**: Reed-Sternberg cells, Ann Arbor staging, types

**Hodgkin Lymphoma**:
- **Reed-Sternberg (RS) cells**: Large binucleated cells with **"owl eye" nucleoli**; CD15+, CD30+; B cell origin
- Spreads **contiguously** (node by node)

| HL Type | RS Cell Variant | Prognosis | Features |
|---------|----------------|----------|---------|
| **Nodular Sclerosis** | **Lacunar cells** | Good | Most common; young women; mediastinal mass |
| **Mixed Cellularity** | Classic RS cells | Intermediate | EBV associated; older men; B symptoms |
| **Lymphocyte Rich** | Few RS cells | **Best** | Rare; young; most favorable |
| **Lymphocyte Depleted** | Many RS cells, few lymph | **Worst** | Elderly; HIV; disseminated |

**Ann Arbor Staging** (HL):
- I: Single lymph node region
- II: ≥2 regions, same side of diaphragm
- III: Both sides of diaphragm
- IV: Extranodal dissemination (bone marrow, liver)
- **A**: No B symptoms; **B**: Fever >38°C, night sweats, >10% weight loss

**Non-Hodgkin Lymphoma** (NHL):
| Type | Age | Genetics | CD Markers | Key Features |
|------|-----|---------|-----------|-------------|
| **Follicular lymphoma** | Middle-aged | **t(14;18) BCL-2** | CD10, CD20, BCL-2+ | Waxing/waning; indolent; **cannot cure** |
| **Burkitt lymphoma** | Children/young adults | **t(8;14) c-MYC** | CD10, CD20 | **Starry sky pattern**; associated with **EBV** (endemic type); jaw mass (Africa) |
| **Diffuse large B-cell (DLBCL)** | Elderly | BCL-6 rearrangement | CD20, CD19 | Most common aggressive NHL; curable with R-CHOP |
| **Mantle cell lymphoma** | Middle-aged men | **t(11;14) cyclin D1** | **CD5+, CD23−** | Aggressive; poor prognosis |
| **Marginal zone (MALT)** | Adults | t(11;18) | CD20 | H. pylori gastric MALT → antibiotics can cure |
| **T-cell lymphoma** | Adults | Variable | CD3, CD4/CD8 | Mycosis fungoides (skin), Sezary syndrome |

### Multiple Myeloma

> **PYQ HOTSPOT**: CRAB criteria, Bence-Jones protein, M spike

- Malignant proliferation of **plasma cells** in bone marrow
- **M protein (M spike)** on SPEP: usually **IgG** (most common), IgA
- **Bence-Jones protein**: Free light chains (κ or λ) in urine

**CRAB criteria**:
- **C** - Hypercalcemia (↑ osteoclast activity)
- **R** - Renal failure (light chain deposition, hypercalcemia)
- **A** - Anemia (normocytic normochromic; marrow replacement)
- **B** - Bone lesions (**"punched-out" lytic lesions**; NO osteoblastic reaction)

**Other features**:
- **Rouleaux formation** on peripheral smear (RBCs stack like coins due to paraprotein)
- **↑ ESR**, ↑ total protein, ↑ globulin
- Recurrent bacterial infections (hypogammaglobulinemia of normal Ig)
- **"Mott cells"** (plasma cells with Russell bodies = Ig-filled vacuoles)

## Cardiovascular Pathology

### Myocardial Infarction

> **PYQ HOTSPOT**: Timeline of changes, complications, zones

**Zones of MI (transmural)**:
1. **Zone of necrosis**: Central dead tissue
2. **Zone of injury**: Surrounding; viable but injured
3. **Zone of ischemia**: Outer; viable, reversible

**Timeline of changes** (CRITICAL - most tested):

| Time | Gross Appearance | Microscopy |
|------|-----------------|-----------|
| **0-4 hours** | Normal (no change visible) | **No change** on LM; electron microscopy: mitochondrial swelling |
| **4-12 hours** | Dark discoloration | **Coagulative necrosis begins**; wavy fibers, edema |
| **12-24 hours** | Dark mottling | **Coagulative necrosis**; **neutrophil infiltration** begins |
| **1-3 days** | Yellow-tan softening | Extensive neutrophils; **pyknosis/karyorrhexis** |
| **3-7 days** | **Maximal softening** (yellow center, red border) | **Macrophages replace neutrophils**; phagocytosis of debris; **GREATEST RISK OF FREE WALL RUPTURE** |
| **1-3 weeks** | Pale/gray; granulation tissue at margins | Granulation tissue (fibroblasts + neovascularization) |
| **>6 weeks** | **White scar** | **Dense fibrous scar** (Type I collagen) |

**Biomarker Timeline**:
| Marker | Rise | Peak | Return to Normal |
|--------|------|------|----------------|
| **Troponin I/T** | 3-4h | **12-24h** | 7-10 days (best for late diagnosis) |
| **CK-MB** | 4-6h | **24h** | 2-3 days (best for reinfarction) |
| **Myoglobin** | **1-2h** (earliest) | 4-6h | 24h |
| LDH | 24h | 48-72h | 10-14 days |

**Complications by Time**:
| Time | Complication |
|------|-------------|
| Day 1-2 | **Arrhythmias** (most common cause of death in 1st 24h); cardiogenic shock |
| Day 2-7 | **Free wall rupture** (cardiac tamponade), papillary muscle rupture (MR), VSD |
| Day 1-3 weeks | **Dressler syndrome** (fibrinous pericarditis, 2-10 weeks; autoimmune) |
| Months | Ventricular aneurysm (persistent ST elevation) |

| Genetics | AD, X-linked (Duchenne) | **AD; MYH7, MYBPC3** | - |
| Cause | Idiopathic, viral, alcohol, peripartum | Genetic mutation | **Amyloid, sarcoid, hemochromatosis, Löffler (eosinophils)** |
| Key feature | "Globular heart", S3 gallop | **Dynamic LVOTO**; sudden death in young athletes; **systolic anterior motion (SAM) of mitral valve** | **Kussmaul sign**; elevated JVP; like constrictive pericarditis |
| Treatment | ACE-I, beta-blocker, ICD | **Avoid digoxin/vasodilators**; beta-blocker, verapamil | Treat cause |

### Rheumatic Heart Disease

- Follows **Group A beta-hemolytic Streptococcus** pharyngitis
- **Immune-mediated** (not direct infection)
- **JONES Criteria**: Joints (migratory polyarthritis), O (heart - carditis), Nodes (subcutaneous), Erythema marginatum, Sydenham chorea

**Histology**: **Aschoff nodules** = focal myocarditis with central fibrinoid necrosis + **Aschoff cells** (activated macrophages, caterpillar/owl-eye nuclei) + **Anitschkow cells**

**Valve involvement** (order of frequency):
1. **Mitral** (most common - mitral stenosis chronic sequela)
2. Aortic
3. Tricuspid
4. Pulmonic (rare)

**Chronic RHD**: Mitral stenosis → "fish mouth/buttonhole" deformity → left atrial enlargement → **AF → systemic emboli**

### Infective Endocarditis

| Feature | Acute | Subacute |
|---------|-------|---------|
| Organism | **Staph aureus** (most common overall) | **Strep viridans** |
| Onset | Rapid (days-weeks) | Slow (weeks-months) |
| Underlying valve | Normal | Abnormal |
| Virulence | High | Low |

**Organisms in special situations**:
- **IV drug users**: Tricuspid valve; **S. aureus**
- **Prosthetic valve early** (<60 days): Staph epidermidis
- **Colon cancer/GI**: Strep bovis (S. gallolyticus)
- **Dental procedure**: Strep viridans
- **Q fever**: Coxiella burnetii (culture negative)
- **HACEK organisms**: Culture negative; gram-negative

**Features**: Osler nodes (finger/toe painful), Janeway lesions (palmar/plantar painless), Roth spots (retinal hemorrhage), splinter hemorrhages, **Libman-Sacks endocarditis** (SLE - sterile, both sides of valve)

### ARDS (Acute Respiratory Distress Syndrome)

- **Definition**: Acute onset, bilateral opacities, PaO2/FiO2 < 300, not cardiogenic
- **Pathology**: **Diffuse alveolar damage (DAD)** → **hyaline membrane formation**
- **Causes**: Sepsis (most common), trauma, aspiration, transfusion
- **Phases**: Exudative (1-7d) → Proliferative (1-3 wks) → Fibrotic
- **Key histology**: Hyaline membranes (fibrin + dead cells lining alveoli)

## GI and Liver Pathology

### Gastric Pathology

**Peptic Ulcer Disease**:
- **H. pylori** → disrupts mucous layer → antral gastritis; **duodenal ulcer** (most common ulcer)
- **NSAID** → inhibit PGE2 → ↓ mucus and bicarb → gastric ulcer
- **Stress ulcers**: Cushing's (CNS, ↑HCl), Curling's (burn, ↓mucosal blood flow)
- **Zollinger-Ellison**: Gastrinoma (MEN1) → ↑↑ gastric acid → multiple severe ulcers

**Gastric Cancer**:

| Type | Lauren Classification | Histology | Features |
|------|---------------------|----------|---------|
| **Intestinal type** | Well-differentiated glands | Glandular; cohesive | Associated with **H. pylori, atrophic gastritis, diet (nitrosamines)**; common in Japan; better prognosis |
| **Diffuse type** | **Signet ring cells** (mucin pushes nucleus) | **Linitis plastica** ("leather bottle stomach") | Associated with **E-cadherin loss (CDH1 mutation)**; young patients; worse prognosis |

**Metastatic patterns**: Krukenberg tumor (ovaries), **Sister Mary Joseph nodule** (umbilical), Virchow node (left supraclavicular)

### Colorectal Cancer

> **PYQ HOTSPOT**: FAP vs HNPCC, pathway of carcinogenesis

| Syndrome | Gene | Pathway | Features |
|---------|------|---------|---------|
| **FAP (Familial Adenomatous Polyposis)** | **APC** | Chromosomal instability; LOH | >100 polyps; **100% cancer risk** without colectomy; Gardner syndrome (osteomas + desmoid) |
| **HNPCC/Lynch Syndrome** | **MLH1, MSH2** (mismatch repair) | **Microsatellite instability (MSI)** | Right-sided CRC; younger; **associated with endometrial, ovarian cancer** |

**Adenoma-Carcinoma Sequence** (Fearon-Vogelstein model):
Normal → **APC loss** → small adenoma → **K-RAS mutation** → large adenoma → **SMAD4, p53 loss** → carcinoma

**Duke's Staging** (simplified):
- A: Mucosa/submucosa
- B: Through muscularis propria
- C: Lymph node involvement
- D: Distant metastasis

### Hepatocellular Carcinoma (HCC)

- **Major risk factors**: **Cirrhosis** (most common), **HBV** (can cause HCC without cirrhosis), HCV, **aflatoxin B1** (HBsAg + aflatoxin = greatest risk), alcohol
- **Tumor marker**: **AFP** (very high = HCC; also elevated in yolk sac tumors)
- **Gross**: Nodular or massive; **satellite nodules** common
- **Microscopy**: Trabecular pattern; bile production by tumor cells (pathognomonic)
- **Fibrolamellar variant**: Young adults; **no cirrhosis**; better prognosis; AFP normal; **lamellar fibrosis**

### Liver Pathology

**Cirrhosis**:
| Type | Nodule Size | Common Causes |
|------|------------|--------------|
| **Micronodular** | <3 mm | **Alcohol** (most common), biliary obstruction, hemochromatosis |
| **Macronodular** | >3 mm | **Viral hepatitis** (HBV, HCV) |
| **Mixed** | Both | Various |

**Portal Hypertension Complications** (mnemonic: **"ABCDE"**):
- **A** - Ascites (most common)
- **B** - Bleeding (varices - esophageal most dangerous)
- **C** - Caput medusae (dilated periumbilical veins)
- **D** - Dupuytren contracture (association)
- **E** - Encephalopathy (hepatic); Edema; splenomegaly

**Hepatitis Histology**:
- **Acute hepatitis**: Hepatocyte ballooning + necrosis, **Councilman bodies** (acidophil/apoptotic hepatocytes), lobular inflammation
- **Chronic hepatitis**: Portal tracts with lymphocytes; **periportal fibrosis** → bridging fibrosis → cirrhosis
- **Alcoholic hepatitis**: **Mallory-Denk bodies** (tangled intermediate filaments), neutrophilic inflammation, fatty change, zone 3 (centrilobular) necrosis
- **Hemochromatosis**: Iron deposits (Prussian blue stain); zone 1 (periportal) first
- **Wilson's disease**: Copper deposits; fatty change; Mallory bodies; **rhodamine stain** for copper
