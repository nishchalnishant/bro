## 8. IMPORTANT DRUG SIDE EFFECTS — MASTER TABLE

| Drug | Classic/Unique Side Effect |
|------|--------------------------|
| **Amiodarone** | Thyroid dysfunction (hypo AND hyper), pulmonary fibrosis, hepatotoxicity, corneal microdeposits, photosensitivity, blue-grey skin discoloration, peripheral neuropathy |
| **Chloroquine/Hydroxychloroquine** | Retinopathy (bull's eye maculopathy, irreversible) |
| **Ethambutol** | Retrobulbar optic neuritis → red-green color blindness |
| **Isoniazid** | Peripheral neuropathy (pyridoxine deficiency), hepatotoxicity, SLE-like syndrome |
| **Rifampicin** | Red-orange discoloration of body fluids, enzyme inducer, hepatotoxicity |
| **Pyrazinamide** | Hyperuricemia/gout, hepatotoxicity |
| **Streptomycin** | Vestibular ototoxicity, nephrotoxicity |
| **Linezolid** | Serotonin syndrome (MAO inhibitor), thrombocytopenia, optic neuropathy |
| **Clozapine** | Agranulocytosis (weekly CBC), seizures, metabolic syndrome |
| **Haloperidol** | EPS (tardive dyskinesia), NMS, QT prolongation |
| **Thioridazine** | Retinal pigmentation, QT prolongation (most cardiotoxic typical AP) |
| **Chlorpromazine** | Photosensitivity, cholestatic jaundice, corneal deposits, lower EPS than haloperidol |
| **Lithium** | Nephrogenic DI, hypothyroidism, tremor, teratogen (Ebstein's anomaly), narrow TI |
| **Valproate** | Neural tube defects (spina bifida), hepatotoxicity, pancreatitis, weight gain, hair loss |
| **Phenytoin** | Gingival hyperplasia, hirsutism, fetal hydantoin syndrome, zero-order kinetics |
| **Carbamazepine** | SIADH, aplastic anemia, Stevens-Johnson (HLA-B*1502), diplopia, enzyme inducer |
| **Methotrexate** | Mucositis, pulmonary pneumonitis, hepatic fibrosis; rescue = leucovorin |
| **Cyclophosphamide** | Hemorrhagic cystitis (acrolein → MESNA), SIADH, infertility |
| **Cisplatin** | Nephrotoxicity, ototoxicity (high-frequency), neuropathy, N/V |
| **Doxorubicin** | Dilated cardiomyopathy (cumulative), red urine |
| **Bleomycin** | Pulmonary fibrosis (cumulative dose), minimal myelosuppression |
| **Vincristine** | Peripheral neuropathy (dose-limiting), minimal myelosuppression, SIADH |
| **Vinblastine** | Myelosuppression (dose-limiting), less neuropathy than vincristine |
| **Busulfan** | Pulmonary fibrosis, hyperpigmentation |
| **Statins (HMG-CoA RI)** | Myopathy/rhabdomyolysis (↑ with CYP3A4 inhibitors), hepatotoxicity, teratogenicity |
| **Fibrates** | Cholelithiasis (↑ cholesterol secretion in bile), myopathy (↑ with statins), hepatotoxicity |
| **Niacin** | Flushing (prostaglandin-mediated, treat with aspirin), hyperglycemia, hyperuricemia, hepatotoxicity |
| **ACE inhibitors** | Dry cough (bradykinin), angioedema, hyperkalemia, teratogenic (renal dysgenesis) |
| **Verapamil** | Constipation, bradycardia, AV block, negative inotropy, gingival hyperplasia |
| **Furosemide** | Hypokalemia, metabolic alkalosis, ototoxicity (IV, rapid), hypocalcemia |
| **Thiazides** | Hypokalemia, hypercalcemia, hyperuricemia, hyperglycemia, hyponatremia |
| **Spironolactone** | Gynecomastia, hyperkalemia, metabolic acidosis |
| **Heparin** | HIT (IgG-PF4 antibody), osteoporosis (long-term), hyperkalemia (aldosterone suppression) |
| **Warfarin** | Bleeding, teratogenic, skin necrosis (protein C deficiency), purple toe syndrome |
| **Tetracyclines** | Teeth staining/enamel hypoplasia in children, photosensitivity, pseudotumor cerebri |
| **Aminoglycosides** | Nephrotoxicity (PCT), ototoxicity (cochlear/vestibular), NMJ blockade |
| **Vancomycin** | Red man syndrome (infusion-related), nephrotoxicity, ototoxicity |
| **Fluoroquinolones** | Tendon rupture (Achilles), cartilage damage in children, QT prolongation, CNS effects |
| **Metronidazole** | Disulfiram-like reaction with alcohol, metallic taste, peripheral neuropathy |
| **Chloramphenicol** | Aplastic anemia (idiosyncratic), gray baby syndrome, dose-related bone marrow suppression |
| **Clindamycin** | Pseudomembranous colitis (C. difficile) |
| **Dapsone** | Hemolytic anemia (G6PD), methemoglobinemia, agranulocytosis |
| **Pyrimethamine** | Folate deficiency (rescue with folinic acid/leucovorin) |
| **Primaquine** | Hemolytic anemia in G6PD deficiency, methemoglobinemia |
| **Mefloquine** | Neuropsychiatric SE (hallucinations, psychosis, vivid dreams) |
| **Amphotericin B** | Nephrotoxicity, infusion reactions (shake and bake), hypokalemia, hypomagnesemia |
| **Griseofulvin** | SLE-like syndrome (ANA positive), photosensitivity, disulfiram-like reaction, teratogenic |
| **Ketoconazole** | Gynecomastia (anti-androgenic — blocks testosterone synthesis), hepatotoxicity, CYP3A4 inhibitor |
| **Tacrolimus/Cyclosporine** | Nephrotoxicity, hypertension, neurotoxicity (tremor); Cyclosporine: gingival hyperplasia, hirsutism; Tacrolimus: DM, alopecia |
| **Mycophenolate** | GI (diarrhea most common), myelosuppression, teratogenic |
| **Allopurinol** | Rash (mild to SJS), hepatotoxicity, interacts with 6-MP/azathioprine (reduce dose to 25%) |
| **Colchicine** | GI (diarrhea most common), myopathy, neuropathy; toxicity: multiorgan failure |
| **Probenecid** | Renal stones, interacts with penicillins (↑ levels — blocks tubular secretion) |
| **Pioglitazone** | Bladder cancer risk, edema, heart failure, fractures |
| **Glitazones** | Weight gain, fluid retention, heart failure, bone fractures |
| **GLP-1 agonists** | N/V, pancreatitis, thyroid C-cell tumors (animal data), injection site reactions |
| **SGLT2 inhibitors** | UTI, genital mycosis, DKA (euglycemic), Fournier's gangrene, polyuria |
| **Metformin** | Lactic acidosis (rare), GI, B12 deficiency, metal taste |
| **Carbimazole/PTU** | Agranulocytosis (most serious) — sore throat = STOP DRUG, check WBC |
| **Lithium** | Tremor, polyuria/polydipsia (NDI), hypothyroidism, Ebstein's anomaly (teratogen), toxicity risk (narrow TI) |
| **Corticosteroids** | Osteoporosis, avascular necrosis (femoral head), Cushingoid, hyperglycemia, posterior subcapsular cataract, immunosuppression |
| **NSAIDs** | GI ulcers (COX-1 inhibition), renal impairment, platelet dysfunction, asthma exacerbation (aspirin-exacerbated), cardiovascular events (COX-2 selective) |

## KEY MNEMONICS SUMMARY

| Mnemonic | Content |
|----------|---------|
| **SLUDGE** | Salivation, Lacrimation, Urination, Defecation, GI distress, Emesis (cholinergic excess) |
| **Dry, Blind, Red, Hot, Mad + Tachy** | Atropine toxicity |
| **A BEAM** | Cardioselective beta-blockers (Atenolol, Bisoprolol, Esmolol, Acebutolol, Metoprolol) |
| **CRAP GPS** | Enzyme inducers (Carbamazepine, Rifampicin, Alcohol, Phenytoin, Griseofulvin, Phenobarbitone, Sulphonylureas) |
| **LAG** | Rapid-acting insulins (Lispro, Aspart, Glulisine) |
| **ACHES** | OCP contraindications |
| **TLC MAN** | First-line TB drugs side effects: (T)uberculosis drugs Liver/optic/CNS problems |
| **3 Cs of TCA OD** | Convulsions, Coma, Cardiac arrhythmias |
