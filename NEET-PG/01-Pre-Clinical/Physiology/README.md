# Physiology

Physiology covers the normal functional mechanisms of the human body — from cellular and membrane physiology to organ system integration — and contributes ~13 questions per NEET PG paper.

## Files in This Folder

| File | Description |
|------|-------------|
| `physiology-notes.md` | Condensed exam-focused notes covering all major organ systems for rapid revision |
| `physiology-lecture.md` | In-depth lecture notes with mechanisms, diagrams, and conceptual explanations |
| `WEAK-TOPICS.md` | Personally flagged weak areas requiring extra revision attention |

## High-Yield Areas

- Cardiovascular physiology: cardiac cycle, pressure-volume loops, Starling's law, Wiggers diagram, and ECG interpretation
- Renal physiology: GFR regulation, tubular transport, countercurrent mechanism, and acid-base homeostasis
- Respiratory physiology: V/Q ratio, oxygen-haemoglobin dissociation curve, lung volumes, and hypoxia types
- Nerve and muscle physiology: action potential generation, neuromuscular junction, types of muscle contraction
- Endocrine physiology: feedback regulation, receptor mechanisms, pituitary-target organ axes, and normal hormonal values
