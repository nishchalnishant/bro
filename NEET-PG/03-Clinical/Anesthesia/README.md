# Anesthesia — NEET PG
**NEET PG Weight:** 4–5 questions (~2%)
**Tier:** 4

## Files in This Folder
- `anesthesia-lecture.md` — First-principles lecture notes for deep understanding (read in Month 8)
- `anesthesia-notes.md` — Condensed exam-focused notes for revision (use from Month 9 onwards)

## Top NEET PG Topics
1. Inhalational agents — MAC values (comparison table), blood-gas partition coefficient, clinical implications
2. Neuromuscular blocking agents — depolarising (suxamethonium: phase I/II block, contraindications) vs non-depolarising (reversal with neostigmine)
3. Regional anesthesia — spinal vs epidural (differences, complications, total spinal, post-dural puncture headache)

## Study Tips
- Anesthesia is a 4–5 question subject; one focused week of revision from the notes file is sufficient.
- MAC value comparison table is almost always tested — isoflurane (1.15%) as reference, N2O (105%), desflurane (6%), sevoflurane (2%).
- Malignant hyperthermia (trigger agents, mechanism, treatment with dantrolene) is a high-yield emergency scenario.
