# Anesthesia

Anesthesia covers the pharmacology of anaesthetic agents, principles of pain management, monitoring, regional techniques, and perioperative care — contributing ~5 questions per NEET PG paper with a slight rising trend.

## Files in This Folder

| File | Description |
|------|-------------|

| `anesthesia-lecture.md` | In-depth lecture notes with MAC values, drug comparisons, and anaesthetic principles |
| `WEAK-TOPICS.md` | Personally flagged weak areas requiring extra revision attention |

## High-Yield Areas

- Inhalational agents: MAC values (isoflurane 1.15% as reference, N2O 105%, desflurane 6%, sevoflurane 2%), blood-gas partition coefficients, and clinical implications
- Neuromuscular blocking agents: suxamethonium (depolarising, Phase I vs Phase II block, contraindications) vs non-depolarising agents (reversal with neostigmine)
- Regional anesthesia: spinal vs epidural differences (onset, spread, catheter use), complications (PDPH management, total spinal), and contraindications
- Malignant hyperthermia: triggering agents (volatile agents and suxamethonium), pathophysiology (RYR1 mutation), and treatment with dantrolene
- Monitoring and equipment: pulse oximetry principles, capnography waveform interpretation, and Guedel's stages of anaesthesia
