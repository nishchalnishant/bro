# Psychiatry

Psychiatry covers the classification, diagnosis, and pharmacological and psychological management of mental disorders — contributing ~6 questions per NEET PG paper.

## Files in This Folder

| File | Description |
|------|-------------|
| `psychiatry-notes.md` | Condensed exam-focused notes covering diagnostic criteria, rating scales, and drug adverse effects |
| `psychiatry-lecture.md` | In-depth lecture notes on psychopathology, nosology (ICD-11/DSM-5), and treatment protocols |
| `WEAK-TOPICS.md` | Personally flagged weak areas requiring extra revision attention |

## High-Yield Areas

- Schizophrenia: Schneider's first-rank symptoms, positive vs negative symptom distinction, antipsychotic drug selection and adverse effects
- Mood disorders: diagnostic criteria for major depression and mania (ICD-11/DSM-5), lithium use, toxicity signs, and monitoring parameters
- Substance use disorders: dependence vs harmful use, withdrawal syndrome features, delirium tremens management, opioid substitution therapy
- Rating scales: Hamilton Depression Rating Scale, MMSE, BPRS, CAGE, AUDIT — which scale measures which construct
- Adverse effects: tardive dyskinesia vs acute dystonia vs NMS vs serotonin syndrome — clinical distinction is a recurring Q
