# Normal Values and Lab Reference — NEET PG Quick Reference

Comprehensive exam-focused reference organized by system. PYQ HOTSPOT markers indicate values directly tested in MCQs.

---

## 1. Hematology

### Hemoglobin (g/dL)

| Group | Normal Range |
|-------|-------------|
| Adult male | 13.5 – 17.5 |
| Adult female | 12.0 – 15.5 |
| Pregnancy | > 11.0 (WHO cutoff) |
| Child (6–12 yr) | 11.5 – 15.5 |
| Neonate (cord blood) | 14 – 20 |
| Neonate at 1 week | 15 – 21 |

> **PYQ HOTSPOT** — WHO anemia cutoffs: male < 13 g/dL, non-pregnant female < 12 g/dL, pregnant female < 11 g/dL, child 6–59 months < 11 g/dL.

---

### WBC Count and Differential

| Parameter | Normal Range |
|-----------|-------------|
| Total WBC | 4,000 – 11,000 /mm³ (4–11 × 10⁹/L) |
| Neutrophils | 40–75% (2,000–7,500 /mm³) |
| Lymphocytes | 20–45% (1,000–4,500 /mm³) |
| Monocytes | 2–10% (100–800 /mm³) |
| Eosinophils | 1–6% (40–400 /mm³) |
| Basophils | 0–1% (< 100 /mm³) |

> **PYQ HOTSPOT** — Absolute eosinophilia: > 500/mm³. Absolute neutrophilia: > 7,500/mm³. Absolute lymphocytosis: > 4,500/mm³.

---

### Platelet Count

| Status | Value |
|--------|-------|
| Normal | 1,50,000 – 4,00,000 /mm³ (1.5–4 × 10¹¹/L) |
| Thrombocytopenia | < 1,50,000 /mm³ |
| Severe thrombocytopenia | < 20,000 /mm³ |
| Thrombocytosis | > 4,00,000 /mm³ |

> **PYQ HOTSPOT** — Spontaneous bleeding risk: platelets < 20,000/mm³. Platelet transfusion trigger in surgery: < 50,000/mm³.

---

### Red Cell Indices

| Index | Normal Range | Interpretation |
|-------|-------------|----------------|
| MCV | 80–100 fL | < 80 = microcytic; > 100 = macrocytic |
| MCH | 27–33 pg | < 27 = hypochromic |
| MCHC | 32–36 g/dL | < 32 = hypochromic; > 36 = spherocytosis |
| RDW | 11.5–14.5% | Elevated = anisocytosis (IDA early) |

> **PYQ HOTSPOT** — In IDA: MCV ↓, MCH ↓, MCHC ↓, RDW ↑. In thalassemia trait: MCV ↓, RDW normal. MCHC > 36 = hereditary spherocytosis.

---

### Erythrocyte Sedimentation Rate (ESR) — Westergren Method

| Group | Normal (mm at 1 hr) |
|-------|---------------------|
| Male < 50 yr | < 15 mm/hr |
| Male > 50 yr | < 20 mm/hr |
| Female < 50 yr | < 20 mm/hr |
| Female > 50 yr | < 30 mm/hr |
| Child | < 10 mm/hr |

> **PYQ HOTSPOT** — ESR is most elevated in multiple myeloma. ESR paradoxically low in sickle cell disease (due to sickling preventing rouleaux).

---

### Reticulocyte Count

| Status | Value |
|--------|-------|
| Normal | 0.5–2.5% of RBCs (25,000–75,000 /mm³) |
| Corrected reticulocyte count | = Observed % × (Patient Hct / 45) |
| Reticulocyte production index (RPI) | RPI < 2 = hypoproliferative; > 3 = blood loss/hemolysis |

---

### Coagulation Studies

| Test | Normal Range |
|------|-------------|
| Prothrombin Time (PT) | 11–13 seconds |
| INR (normal) | 0.8–1.2 |
| INR (therapeutic — AF, DVT) | 2.0–3.0 |
| INR (mechanical heart valve) | 2.5–3.5 |
| aPTT | 25–40 seconds |
| Bleeding Time (Ivy) | 2–9 minutes |
| Clotting Time (Lee-White) | 5–15 minutes |
| Thrombin time | 14–19 seconds |
| Fibrinogen | 200–400 mg/dL |
| D-dimer | < 0.5 mg/L (< 500 ng/mL) |

> **PYQ HOTSPOT** — PT/INR prolonged: warfarin, liver disease, Vit K deficiency, Factor VII deficiency (extrinsic pathway). aPTT prolonged: heparin, hemophilia A/B, Factor VIII/IX deficiency (intrinsic pathway). Both prolonged: common pathway defects, DIC.

## 2. Biochemistry / Metabolic

### Serum Electrolytes

| Electrolyte | Normal Range |
|-------------|-------------|
| Sodium (Na⁺) | 136–145 mEq/L |
| Potassium (K⁺) | 3.5–5.0 mEq/L |
| Chloride (Cl⁻) | 98–106 mEq/L |
| Bicarbonate (HCO₃⁻) | 22–29 mEq/L |
| Calcium (Ca²⁺) — total | 8.5–10.5 mg/dL (2.1–2.6 mmol/L) |
| Calcium (Ca²⁺) — ionized | 4.6–5.3 mg/dL (1.15–1.35 mmol/L) |
| Magnesium (Mg²⁺) | 1.8–2.6 mg/dL (0.75–1.0 mmol/L) |
| Phosphate (inorganic) | 2.5–4.5 mg/dL (0.81–1.45 mmol/L) |

> **PYQ HOTSPOT** — Corrected calcium = Measured Ca + 0.8 × (4 – measured albumin). Anion gap = Na – (Cl + HCO₃) = 8–12 mEq/L (normal); > 12 = high AG metabolic acidosis (MUDPILES).

---

### Glucose and Diabetes Diagnosis

| Parameter | Normal | Pre-diabetes | Diabetes |
|-----------|--------|-------------|---------|
| Fasting plasma glucose | < 100 mg/dL | 100–125 mg/dL | ≥ 126 mg/dL |
| 2-hr OGTT (75g) | < 140 mg/dL | 140–199 mg/dL | ≥ 200 mg/dL |
| HbA1c | < 5.7% | 5.7–6.4% | ≥ 6.5% |
| Random plasma glucose (symptomatic) | — | — | ≥ 200 mg/dL |

> **PYQ HOTSPOT** — ADA 2023 diagnostic criteria require 2 abnormal tests unless classic symptoms + random glucose ≥ 200 mg/dL. HbA1c target in treated diabetes: < 7% (ADA); < 6.5% (AACE).

---

### Renal Function Tests

| Parameter | Normal Range |
|-----------|-------------|
| Serum creatinine (male) | 0.7–1.3 mg/dL |
| Serum creatinine (female) | 0.6–1.1 mg/dL |
| BUN (Blood Urea Nitrogen) | 7–20 mg/dL |
| Serum urea | 15–45 mg/dL |
| BUN:Creatinine ratio | 10:1 – 20:1 (> 20:1 = prerenal) |
| Uric acid (male) | 3.5–7.2 mg/dL |
| Uric acid (female) | 2.6–6.0 mg/dL |

> **PYQ HOTSPOT** — Gout diagnosis: serum uric acid > 6.8 mg/dL. Uricosuric drugs (probenecid) work best when uric acid is high-secretion; allopurinol (XO inhibitor) is first line for overproducers.

---

### Serum Proteins

| Parameter | Normal Range |
|-----------|-------------|
| Total protein | 6.4–8.3 g/dL |
| Serum albumin | 3.5–5.0 g/dL |
| Globulin | 2.0–3.5 g/dL |
| A:G ratio | 1.5:1 – 2.5:1 |

> **PYQ HOTSPOT** — Albumin < 3.5 g/dL = hypoalbuminemia (malnutrition, cirrhosis, nephrotic syndrome). Reversed A:G ratio (< 1) in multiple myeloma, liver disease, chronic infections.

---

### Liver Function Tests (LFTs)

| Parameter | Normal Range |
|-----------|-------------|
| Total bilirubin | 0.3–1.2 mg/dL |
| Direct (conjugated) bilirubin | 0.0–0.3 mg/dL |
| Indirect (unconjugated) bilirubin | 0.2–0.9 mg/dL |
| ALT (SGPT) | 7–56 U/L |
| AST (SGOT) | 10–40 U/L |
| Alkaline phosphatase (ALP) | 44–147 U/L (adult) |
| GGT (Gamma-GT) | Male: 9–48 U/L; Female: 5–36 U/L |
| LDH | 140–280 U/L |

> **PYQ HOTSPOT** — AST:ALT ratio > 2:1 = alcoholic liver disease. ALT > AST = viral hepatitis. ALP markedly elevated = cholestasis or bone disease. GGT elevated in alcoholism (most sensitive marker). Jaundice appears clinically when bilirubin > 2.5–3 mg/dL.

---

### Lipid Profile

| Parameter | Desirable | Borderline | High Risk |
|-----------|-----------|------------|-----------|
| Total cholesterol | < 200 mg/dL | 200–239 mg/dL | ≥ 240 mg/dL |
| LDL | < 100 mg/dL (optimal) | 130–159 mg/dL | ≥ 160 mg/dL |
| HDL | ≥ 60 mg/dL (protective) | 40–59 mg/dL | < 40 mg/dL (risk) |
| Triglycerides | < 150 mg/dL | 150–199 mg/dL | ≥ 200 mg/dL (high) |

**Treatment targets (ATP III / ACC-AHA):**
- Very high risk (ACS, DM + CAD): LDL < 70 mg/dL
- High risk (CAD equivalent): LDL < 100 mg/dL
- Hypertriglyceridemia treatment: TG > 500 mg/dL (pancreatitis risk — fibrates first line)

> **PYQ HOTSPOT** — HDL < 40 mg/dL in males is an independent risk factor for CAD. Non-HDL cholesterol = Total cholesterol – HDL (target: LDL target + 30 mg/dL).

---

### Pancreatic Enzymes

| Parameter | Normal Range | Elevation Significance |
|-----------|-------------|----------------------|
| Serum amylase | 28–100 U/L | > 3× ULN = acute pancreatitis |
| Serum lipase | 0–160 U/L | > 3× ULN = more specific than amylase |

> **PYQ HOTSPOT** — Lipase is more specific than amylase for pancreatitis. Amylase returns to normal in 3–5 days; lipase stays elevated for 7–14 days. Amylase elevated in parotitis, bowel ischemia, renal failure.

## 3. Thyroid and Endocrine

### Thyroid Function

| Parameter | Normal Range |
|-----------|-------------|
| TSH | 0.4–4.5 mIU/L |
| Free T4 (FT4) | 0.8–1.8 ng/dL |
| Free T3 (FT3) | 2.3–4.2 pg/mL |
| Total T4 | 5.0–12.0 µg/dL |
| Total T3 | 80–200 ng/dL |

> **PYQ HOTSPOT** — Subclinical hypothyroidism: TSH ↑, FT4 normal. Subclinical hyperthyroidism: TSH ↓ (suppressed), FT4 normal. Primary hypothyroidism: TSH ↑, FT4 ↓. Sick euthyroid: all low (T3 falls first). Treat Hashimoto's when TSH > 10 mIU/L even if asymptomatic.

---

### Adrenal / Pituitary

| Parameter | Normal Range |
|-----------|-------------|
| Cortisol (morning, 8 AM) | 6–23 µg/dL (166–635 nmol/L) |
| ACTH (morning) | 10–60 pg/mL |
| Cortisol (midnight) | < 1.8 µg/dL |
| Dexamethasone suppression (overnight 1 mg) | Cortisol < 1.8 µg/dL (normal suppression) |
| GH (fasting, random) | < 5 ng/mL (variable) |
| IGF-1 | Age-dependent; 116–358 ng/mL (20–30 yr) |
| Prolactin (male) | 2–18 ng/mL |
| Prolactin (female, non-pregnant) | 2–29 ng/mL |

> **PYQ HOTSPOT** — Prolactin > 200 ng/mL = prolactinoma almost certain. Prolactin 20–200 ng/mL = any hyperprolactinemia cause. Best screening test for Cushing's = 24-hr urinary free cortisol or overnight 1 mg DST. ACTH-dependent Cushing's: ACTH ↑; ACTH-independent: ACTH ↓.

---

### Glucose-Insulin Axis

| Parameter | Normal Range |
|-----------|-------------|
| Fasting insulin | 2–25 µIU/mL |
| C-peptide (fasting) | 0.5–2.0 ng/mL |
| HOMA-IR | < 2.0 (insulin resistance if > 2.5) |

> **PYQ HOTSPOT** — C-peptide low in Type 1 DM and exogenous insulin factitious hypoglycemia. C-peptide elevated in insulinoma and sulfonylurea use. C-peptide is not affected by exogenous insulin assay.

## 4. Arterial Blood Gas (ABG)

### Normal ABG Values

| Parameter | Normal Range |
|-----------|-------------|
| pH | 7.35–7.45 |
| PaO₂ | 80–100 mmHg |
| PaCO₂ | 35–45 mmHg |
| HCO₃⁻ (actual) | 22–26 mEq/L |
| Base excess (BE) | -2 to +2 mEq/L |
| SpO₂ | 95–100% |
| O₂ saturation (arterial) | 95–100% |

> **PYQ HOTSPOT** — pH < 7.35 = acidosis; pH > 7.45 = alkalosis. PaCO₂ > 45 = hypoventilation (respiratory acidosis or compensation for metabolic alkalosis). PaO₂ < 60 mmHg = hypoxia (need O₂ therapy).

---

### ABG Pattern Interpretation

| Disorder | pH | PaCO₂ | HCO₃⁻ | Compensation |
|----------|-----|--------|--------|-------------|
| Metabolic acidosis | ↓ | ↓ (comp) | ↓ (primary) | PaCO₂ = 1.5×HCO₃ + 8 ± 2 (Winter's) |
| Metabolic alkalosis | ↑ | ↑ (comp) | ↑ (primary) | PaCO₂ increases 0.7 per 1 mEq HCO₃ rise |
| Respiratory acidosis (acute) | ↓ | ↑ (primary) | ↑ 1 per 10 mmHg CO₂ | HCO₃ rises 1 mEq/L per 10 mmHg CO₂ |
| Respiratory acidosis (chronic) | ↓ (less) | ↑ (primary) | ↑ 3.5 per 10 mmHg CO₂ | HCO₃ rises 3.5 mEq/L per 10 mmHg CO₂ |
| Respiratory alkalosis (acute) | ↑ | ↓ (primary) | ↓ 2 per 10 mmHg CO₂ | HCO₃ falls 2 mEq/L per 10 mmHg CO₂ |
| Respiratory alkalosis (chronic) | ↑ (less) | ↓ (primary) | ↓ 5 per 10 mmHg CO₂ | HCO₃ falls 5 mEq/L per 10 mmHg CO₂ |

> **PYQ HOTSPOT** — Most common cause of metabolic acidosis with high AG = lactic acidosis. Most common cause of metabolic alkalosis = vomiting (loss of HCl). Type 1 RTA: normal AG acidosis, urine pH > 5.5. Type 4 RTA: hyperkalemia + normal AG acidosis (hypoaldosteronism).

## 5. Renal

### GFR and CKD Staging

| CKD Stage | eGFR (mL/min/1.73m²) | Description |
|-----------|----------------------|-------------|
| G1 | ≥ 90 | Normal or high (with kidney damage markers) |
| G2 | 60–89 | Mildly decreased |
| G3a | 45–59 | Mildly-moderately decreased |
| G3b | 30–44 | Moderately-severely decreased |
| G4 | 15–29 | Severely decreased |
| G5 | < 15 | Kidney failure (dialysis/transplant) |

> **PYQ HOTSPOT** — KDIGO 2012: CKD defined as GFR < 60 OR kidney damage markers for > 3 months. Dialysis generally started at GFR < 10 (symptomatic) or < 5–7 (asymptomatic). Erythropoietin deficiency anemia starts when GFR < 30.

---

### Creatinine Clearance — Cockcroft-Gault Formula

**CrCl (mL/min) = [(140 – age) × weight (kg)] / [72 × serum creatinine (mg/dL)]**
- Multiply by **0.85 for females**

> **PYQ HOTSPOT** — Cockcroft-Gault used for drug dosing. MDRD or CKD-EPI used for GFR estimation and CKD staging. CrCl overestimates GFR (tubular secretion of creatinine).

---

### Urinalysis

| Parameter | Normal |
|-----------|--------|
| Urine specific gravity | 1.003–1.030 |
| Urine pH | 4.6–8.0 |
| Urine osmolality | 50–1,200 mOsm/kg |
| Serum osmolality | 275–295 mOsm/kg |
| Urine protein (24-hour) | < 150 mg/day |
| Urine protein (spot) | < 30 mg/mmol creatinine (ACR) |
| Microalbuminuria | 30–300 mg/day (AER) or ACR 30–300 mg/g |
| Macroalbuminuria (nephrotic range) | > 3,500 mg/day (> 3.5 g/day) |

> **PYQ HOTSPOT** — Nephrotic syndrome: proteinuria > 3.5 g/day + hypoalbuminemia + edema + hyperlipidemia. Nephritic syndrome: hematuria + proteinuria < 3 g/day + hypertension + oliguria + RBC casts.

## 6. Cardiac

### ECG Intervals

| Interval | Normal Duration | Abnormal |
|----------|----------------|---------|
| PR interval | 120–200 ms (3–5 small squares) | > 200 ms = 1° AV block |
| QRS duration | < 120 ms (< 3 small squares) | > 120 ms = BBB or aberrant |
| QTc (Bazett) | Male ≤ 440 ms; Female ≤ 460 ms | > 500 ms = high TdP risk |
| QT interval | 350–440 ms | Varies with heart rate |
| RR interval | 0.6–1.0 s (HR 60–100 bpm) | — |

> **PYQ HOTSPOT** — Bazett formula: QTc = QT / √RR (in seconds). QTc prolongation causes: hypokalemia, hypomagnesemia, hypocalcemia, drugs (amiodarone, sotalol, haloperidol, erythromycin, chloroquine). Delta wave (short PR + wide QRS) = WPW syndrome.

---

### Cardiac Biomarkers in MI

| Marker | Rises | Peaks | Returns to Normal |
|--------|-------|-------|------------------|
| Troponin I | 3–6 hrs | 12–24 hrs | 7–10 days |
| Troponin T | 3–6 hrs | 12–48 hrs | 14 days |
| CK-MB | 3–6 hrs | 12–24 hrs | 48–72 hrs |
| Myoglobin | 1–3 hrs | 4–8 hrs | 12–24 hrs |
| LDH | 6–10 hrs | 24–72 hrs | 7–14 days |

> **PYQ HOTSPOT** — Myoglobin = earliest marker. Troponin = most sensitive and specific (gold standard). CK-MB: reinfarction detection (returns to normal fastest of the specific markers). LDH1:LDH2 ratio > 1 ("flipped") = MI (historical). Troponin can remain elevated 14 days — cannot diagnose reinfarction within this window; use CK-MB instead.

---

### Heart Failure Markers

| Marker | Heart Failure Excluded | Heart Failure Likely |
|--------|------------------------|---------------------|
| BNP | < 100 pg/mL | > 400 pg/mL |
| NT-proBNP (< 50 yr) | < 300 pg/mL | > 450 pg/mL |
| NT-proBNP (50–75 yr) | < 300 pg/mL | > 900 pg/mL |
| NT-proBNP (> 75 yr) | < 300 pg/mL | > 1,800 pg/mL |

> **PYQ HOTSPOT** — BNP is released from ventricles in response to pressure/volume overload. NT-proBNP has longer half-life (better for outpatient monitoring). Both elevated in HF, PE, RV strain, and CKD (reduced clearance).

### Normal CSF Values

| Appearance | Clear and colorless |
| Cell count | 0–5 lymphocytes/mm³ |
| Protein | 15–45 mg/dL (lumbar); 10–25 mg/dL (ventricular) |

| Chloride | 120–130 mEq/L |

---

### CSF Patterns in Meningitis

| Feature | Bacterial | TB | Viral | Cryptococcal |
|---------|-----------|-----|-------|-------------|
| Appearance | Turbid/purulent | Turbid, fibrin web | Clear | Clear/turbid |
| Pressure | ↑↑ | ↑ | Normal/↑ | ↑↑ |
| Cells (/mm³) | 200–20,000 PMNs | 100–500 lymphocytes | 10–1000 lymphocytes | 10–200 lymphocytes |
| Protein (mg/dL) | 100–500 ↑↑ | 100–500 ↑↑ | 50–150 mild ↑ | 50–200 ↑ |
| Glucose | ↓↓ (< 45 mg/dL) | ↓↓ | Normal (rarely ↓) | ↓ |
| CSF:serum glucose | < 0.3 | < 0.5 | ≥ 0.6 | < 0.5 |
| Specific finding | — | Fibrin web, AFB | — | India ink (+), Ag (+) |

> **PYQ HOTSPOT** — Most common cause of bacterial meningitis in adults: S. pneumoniae. In neonates: GBS, E. coli, Listeria. Kernig's sign = inability to extend knee when hip flexed. Brudzinski's sign = neck flexion causes involuntary knee flexion. Xanthochromia in CSF = subarachnoid hemorrhage.

## 8. Obstetrics and Pediatrics

### Fetal Monitoring

| Parameter | Normal |
|-----------|--------|
| Fetal heart rate | 110–160 bpm |
| Baseline variability | 6–25 bpm (moderate variability = reassuring) |
| Acceleration | FHR rises ≥ 15 bpm for ≥ 15 seconds |
| Late deceleration | Uteroplacental insufficiency (non-reassuring) |

> **PYQ HOTSPOT** — FHR < 110 = fetal bradycardia; > 160 = tachycardia. Sinusoidal pattern = severe fetal anemia or asphyxia. Reactive NST = 2 accelerations in 20 minutes (reassuring).

---

### Bishop Score

| Component | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
| Dilation (cm) | Closed | 1–2 | 3–4 | ≥ 5 |
| Effacement (%) | 0–30 | 40–50 | 60–70 | ≥ 80 |
| Station | -3 | -2 | -1/0 | +1/+2 |
| Consistency | Firm | Medium | Soft | — |
| Position | Posterior | Mid | Anterior | — |

> **PYQ HOTSPOT** — Bishop score ≥ 8 = favorable cervix (induction likely successful). Bishop score ≤ 6 = cervical ripening needed before induction. Maximum Bishop score = 13.

---

### APGAR Score

| Component | 0 | 1 | 2 |
|-----------|---|---|---|
| Appearance (color) | Blue/pale all over | Blue extremities, pink body | Pink all over |
| Pulse (heart rate) | Absent | < 100 bpm | ≥ 100 bpm |
| Grimace (reflex) | No response | Grimace | Cry/cough/sneeze |
| Activity (tone) | Limp | Some flexion | Active motion |
| Respiration | Absent | Weak/irregular | Strong cry |

**Timing:** Assessed at 1 minute and 5 minutes after birth.

| Score | Interpretation | Action |
|-------|----------------|--------|
| 7–10 | Normal | Routine care |
| 4–6 | Moderate depression | Stimulate, O₂, monitor |
| 0–3 | Severe depression | Immediate resuscitation |

> **PYQ HOTSPOT** — 1-min APGAR reflects intrapartum asphyxia; 5-min APGAR predicts neonatal outcome. APGAR is not used to decide resuscitation (start resuscitation before 1-min score). Mnemonic: **A**ppearance **P**ulse **G**rimace **A**ctivity **R**espiration.

---

### Birth Weight Milestones

| Milestone | Age |
|-----------|-----|
| Birth weight doubles | 5 months |
| Birth weight triples | 1 year |
| Birth weight quadruples | 2 years |
| Birth length doubles | 4 years |
| Birth weight (average) | 2.5–3.5 kg (LBW < 2.5 kg) |
| Head circumference at birth | 33–35 cm |
| Chest circumference at birth | 30–33 cm (head > chest at birth) |

> **PYQ HOTSPOT** — Head circumference = chest circumference at 6 months (equalization). Chest > head after 6 months. LBW = < 2.5 kg. VLBW = < 1.5 kg. ELBW = < 1 kg.

---

### Developmental Milestones (Key Ages)

| Age | Gross Motor | Fine Motor | Language | Social |
|-----|------------|------------|----------|--------|
| 3 months | Holds head at 45° prone | Hands unfisted, swipes at objects | Cooing | Social smile (6 wk onset) |
| 6 months | Sits with support, rolls over | Palmar grasp, transfers hand-to-hand | Babbling, razzing | Stranger anxiety begins |
| 9 months | Sits without support, stands with support | Pincer grasp (immature) | "Mama/dada" non-specific | Stranger anxiety peak |
| 12 months | Walks with support (cruises), may walk alone | Mature pincer grasp | 1–2 meaningful words | Waves bye-bye, plays pat-a-cake |
| 18 months | Walks well, runs clumsily | Scribbles, tower of 3–4 cubes | 10–15 words, 2-word phrases | Parallel play, feeds self |
| 24 months | Runs well, up/down stairs (2 feet/step) | Tower of 6–7 cubes, copies lines | 50+ words, 2-word sentences | Parallel → associative play |

> **PYQ HOTSPOT** — Red flags: no social smile by 3 months, no babbling by 12 months, no single words by 16 months, no 2-word phrases by 24 months. Loss of milestones at any age = immediate evaluation. Social smile: 6 weeks (earliest tested).

## 9. Key Clinical Cutoffs (Frequently Tested)

### Body Mass Index (BMI)

| Category | BMI (kg/m²) |
|----------|------------|
| Underweight | < 18.5 |
| Normal | 18.5–24.9 |
| Overweight | 25.0–29.9 |
| Obese Class I | 30.0–34.9 |
| Obese Class II | 35.0–39.9 |
| Obese Class III (morbid) | ≥ 40.0 |

> **PYQ HOTSPOT** — Asian-specific cutoffs (WHO modified): overweight ≥ 23 kg/m², obese ≥ 27.5 kg/m². Bariatric surgery indicated: BMI ≥ 40 or BMI ≥ 35 with comorbidities. Waist circumference risk: male > 102 cm (> 90 cm in Asians); female > 88 cm (> 80 cm in Asians).

---

### Blood Pressure Classification (ACC/AHA 2017)

| Category | Systolic (mmHg) | | Diastolic (mmHg) |
|----------|-----------------|---|-----------------|
| Normal | < 120 | and | < 80 |
| Elevated | 120–129 | and | < 80 |
| HTN Stage 1 | 130–139 | or | 80–89 |
| HTN Stage 2 | ≥ 140 | or | ≥ 90 |
| Hypertensive crisis | > 180 | and/or | > 120 |

> **PYQ HOTSPOT** — Old JNC 7 cutoff (still tested): HTN = BP ≥ 140/90. ACC/AHA 2017 lowered to 130/80. Hypertensive urgency = severe BP without end-organ damage. Hypertensive emergency = severe BP with end-organ damage (treat with IV labetalol/nicardipine).

### Child-Pugh Score (Liver Cirrhosis Severity)

| Parameter | 1 point | 2 points | 3 points |
|-----------|---------|---------|---------|

**Interpretation:**

| Class | Total Score | 1-year survival | 2-year survival |
