# Common Anatomy Error Patterns

## Errors That Recur In Tests
- Remembering muscle but not nerve supply.
- Remembering nerve but not root value.
- Knowing structure but forgetting relation.
- Knowing embryology fact but not clinical anomaly.
- Recognizing histology organ family but not exact slide.

## Fix Method
- For every missed question, add one anchor fact and one differentiator.
- Example:
  spleen has central arteriole, lymph node does not.
- Example:
  carpal tunnel does not contain flexor carpi radialis tendon inside the tunnel proper in the same way candidates often assume.

## Weekly Mistake Review
- Revisit all nerve injury mistakes twice weekly.
- Revisit all histology image mistakes once weekly.
- Revisit all embryology mistakes before every grand test.
