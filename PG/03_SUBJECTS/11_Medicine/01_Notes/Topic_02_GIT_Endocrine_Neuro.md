# GIT, Endocrine, And Neurology

## GIT And Liver
- GI bleed, pancreatitis, hepatitis, cirrhosis, portal hypertension, IBD, and malabsorption.

## Endocrine
- Diabetes, DKA, HHS, thyroid disorders, adrenal disorders, pituitary disease, calcium disorders.

## Neurology
- Stroke, seizures, meningitis, neuropathy, myopathy, movement disorders, and coma.

## Must Learn
- Liver function pattern recognition.
- DKA vs HHS.
- Stroke localization.
- Seizure drug choices.
