# Common Biochemistry Error Patterns: The Rate-Limiting Trap

Biochemistry is a memorization-heavy subject where one wrong enzyme name can cost a mark. Use this guide to prevent common errors.

## 1. Rate-Limiting Enzyme Master List (The #1 Error Source)
| Pathway | Rate-Limiting Enzyme |
| :--- | :--- |
| **Glycolysis** | **PFK-1** (not Hexokinase) |
| **Gluconeogenesis** | **Fructose-1,6-bisphosphatase** |
| **TCA Cycle** | **Isocitrate Dehydrogenase** |
| **Glycogenesis** | **Glycogen Synthase** |
| **Glycogenolysis** | **Glycogen Phosphorylase** |
| **HMP Shunt** | **G6PD** |
| **FA Synthesis** | **Acetyl-CoA Carboxylase** (ACC) |
| **FA Oxidation** | **CAT-1 / CPT-1** |
| **Cholesterol Synthesis** | **HMG-CoA Reductase** |
| **Urea Cycle** | **CPS-1** |
| **Heme Synthesis** | **ALA Synthase** |

## 2. Competitive vs Non-competitive Confusion
- **Error**: Thinking Vmax changes in Competitive.
- **Fact**: Vmax stays the same; only **Km increases**. Think of it as needing more substrate to "compete away" the inhibitor.

## 3. B12 vs Folate Deficiency
- **Error**: Diagnosing B12 deficiency based on Megaloblastic Anemia alone.
- **Fact**: Both cause megaloblastic anemia, but **B12 ONLY** causes neuro symptoms (SCDC) and elevated **Methylmalonic Acid (MMA)** levels. Folate only has high Homocysteine.

## 4. G6PD Deficiency Morphology
- **Error**: Confusing Heinz bodies with Howell-Jolly bodies.
- **Fact**: **Heinz bodies** (Denatured Hb) are seen in G6PD. **Howell-Jolly bodies** (DNA remnants) are seen in Post-splenectomy.

## 5. Amino Acid Precursors
- **Error**: Confusing Tryptophan with Tyrosine.
- **Fact**: 
    - **Tryptophan** -> Niacin, Serotonin, Melatonin.
    - **Tyrosine** -> Catecholamines (Dopamine, Epi), Melanin, Thyroid hormone.
- Remembering disease but not enzyme.
- Remembering vitamin but not cofactor role.
- Confusing technique name with application.
- Mixing unconjugated and conjugated bilirubin patterns.
