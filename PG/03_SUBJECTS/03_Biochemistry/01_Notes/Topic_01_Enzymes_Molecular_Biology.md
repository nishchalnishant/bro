# Enzymes And Molecular Biology

## 1. Enzyme Kinetics & Inhibition

### Michaelis-Menten Kinetics
- **Km (Michaelis Constant)**: Substrate concentration at which velocity is half-maximal. Inversely proportional to affinity.
- **Vmax**: Maximum velocity when all enzyme is saturated.

### Inhibition Master Table (Very High Yield)
| Inhibition Type | Vmax | Km | Example |
| :--- | :--- | :--- | :--- |
| **Competitive** | Unchanged | **Increased** | Statins (HMG-CoA Reductase), Methotrexate |
| **Non-competitive** | **Decreased** | Unchanged | Heavy metals (Lead, Mercury) |
| **Uncompetitive** | Decreased | Decreased | Lithium (Inositol monophosphatase) |

- **Suicidal Inhibition**: Enzyme converts the substrate into an irreversible inhibitor (e.g., Allopurinol -> Alloxanthine, Aspirin).

## 2. Molecular Biology

### DNA Replication
- **Orientation**: Occurs in **5' -> 3'** direction.
- **Topoisomerase**: Prevents supercoiling (Fluoroquinolones inhibit bacterial Gyrase).
- **DNA Polymerase III**: Main replicative enzyme in prokaryotes.
- **Telomerase**: RNA-dependent DNA polymerase; maintains chromosome ends in eukaryotes.

### DNA Repair Mechanisms & Disorders
| Mechanism | Defect Causes... | Key Enzyme / Feature |
| :--- | :--- | :--- |
| **Nucleotide Excision (NER)** | **Xeroderma Pigmentosum** | Endonucleases; UV damage |
| **Base Excision (BER)** | - | DNA Glycosylase; Spontaneous deamination |
| **Mismatch Repair** | **Lynch Syndrome (HNPCC)** | MutS/MutL homologs |
| **Homologous Recombination** | **Fanconi Anemia / BRCA1** | Double-strand breaks |

### Genetic Code Characteristics
- **Degeneracy**: Multiple codons for one amino acid.
- **Universal**: Same code for almost all organisms (Exception: Mitochondria).
- **Wobble Hypothesis**: Base pairing at the 3rd position of the codon is flexible.

## 3. Protein Synthesis (Translation)
- **Initiation**: **AUG** (Methionine).
- **Termination**: UAA, UAG, UGA (**U** **A**re **A**way, **U** **A**re **G**one, **U** **G**o **A**way).
- **Inhibitors (Antibiotics)**:
    - **30S**: Aminoglycosides (Streptomycin), Tetracyclines.
    - **50S**: Chloramphenicol, Erythromycin (Macrolides), Clindamycin.

## 4. One-Liners & Clinical Correlation
- **Isoenzymes**: Different forms of the same enzyme (e.g., LDH1 in heart, LDH5 in liver).
- **PCR Stages**: Denaturation (95C) -> Annealing (55C) -> Extension (72C).
- **Blotting Mnemonic**: **S N O W  D R O P**
    - **S**outhern = **D**NA.
    - **N**orthern = **R**NA.
    - **O** = O.
    - **W**estern = **P**rotein.
- **Z-DNA**: Left-handed helix; seen in high salt concentrations.
