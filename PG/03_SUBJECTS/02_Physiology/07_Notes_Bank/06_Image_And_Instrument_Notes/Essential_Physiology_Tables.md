# Essential Physiology Tables

These master tables are designed for rapid comparison of related physiological concepts.

## 1. Fluid Compartments Comparison
| Compartment | % Body Weight | Main Cation | Main Anion |
| :--- | :--- | :--- | :--- |
| **Total Body Water** | 60% | - | - |
| **Intracellular (ICF)** | 40% | **K+**, Mg2+ | Phosphate, Proteins |
| **Extracellular (ECF)** | 20% | **Na+** | Cl-, HCO3- |
| **Plasma** | 5% | Na+ | Cl-, HCO3- (Low protein) |

## 2. G-Protein Coupled Receptors (GPCR)
| Pathway | Secondary Messenger | Hormones / Receptors |
| :--- | :--- | :--- |
| **Gs** | Increased cAMP | Beta-1, Beta-2, Glucagon, PTH, TSH, FSH, LH |
| **Gi** | Decreased cAMP | Alpha-2, M2, D2 |
| **Gq** | IP3 / DAG / Ca2+ | Alpha-1, M1, M3, Oxytocin, GnRH, TRH |

## 3. Lung Disease Patterns
| Parameter | Obstructive (Asthma/COPD) | Restrictive (Fibrosis) |
| :--- | :--- | :--- |
| **FEV1** | Decreased (Markedly) | Decreased |
| **FVC** | Decreased | Decreased (Markedly) |
| **FEV1/FVC Ratio** | **Decreased (< 0.7)** | **Normal or Increased** |
| **TLC / RV** | **Increased** | **Decreased** |

## 4. Nerve Fiber Classification
| Type | Myelination | Function | Speed |
| :--- | :--- | :--- | :--- |
| **A-alpha** | High | Proprioception, Somatic motor | 70-120 m/s |
| **A-delta** | Low | Fast pain, Cold temperature | 12-30 m/s |
| **B** | Low | Pre-ganglionic autonomic | 3-15 m/s |
| **C** | **None** | Slow pain, Heat, Post-ganglionic | 0.5-2 m/s |

## 5. Gastrointestinal Hormones
| Hormone | Source | Stimulus | Primary Action |
| :--- | :--- | :--- | :--- |
| **Gastrin** | G-cells | Distension, Peptides | Increases H+ secretion |
| **Secretin** | S-cells | Acid in Duodenum | Increases Bicarbonate |
| **CCK** | I-cells | Fat / Proteins | SB contraction, Enzymes |
| **Somatostatin** | D-cells | Acid | Inhibits ALL gut hormones |

## 6. JVP Waveforms Interpretation
- **a wave**: Atrial contraction. Absent in AFib. Giant in Tricuspid Stenosis.
- **c wave**: Right ventricular contraction (tricuspid bulging).
- **v wave**: Passive atrial filling. Large in Tricuspid Regurgitation.
- **y descent**: Atrial emptying. Rapid in Constrictive Pericarditis.
- Hormones and second messengers.
- Lung volumes and capacities.
- Acid-base primary disorders.
- ECG interval normal values.
- Renal clearance markers.
