# Microbiology Visual Master Key: Colonies & Morphologies

Microbiology is visually intensive. Use this master key to recognize diagnostic patterns at a glance.

## 1. Bacterial Colony Morphologies (Visual Keys)

| Key Feature | Associated Organism | Clinical Relevance |
| :--- | :--- | :--- |
| **Swarming** | *Proteus mirabilis* | Concentric rings on agar; UTI clue |
| **Medusa Head** | *Bacillus anthracis* | Tangled curls on agar; Anthrax |
| **Bamboo Stick** | *Bacillus anthracis* | Long chains of bacilli on Gram stain |
| **Draughtsman** | *Streptococcus pneumoniae* | Central umbonate depression (Hemolysis) |
| **Black Colonies** | *C. diphtheriae* | On Potassium Tellurite medium |
| **Mercury Drops** | *Bordetella pertussis* | On Regan-Lowe / Bordet-Gengou medium |
| **Satellitism** | *H. influenzae* | Growth around *S. aureus* colonies |
| **Ground Glass** | *Legionella* | On BCYE agar (Enriched) |
| **Fried Egg** | *Mycoplasma* | Central nipple-like growth |
| **Dinger's Ring** | *Leptospira* | Opalescent ring in semi-solid medium |

## 2. Parasitology Egg Identification Key

| Egg Type | Description | Image Clue |
| :--- | :--- | :--- |
| **Ascaris** | Round/Oval, thick mamillated coat | Bile-stained (Brown) |
| **Hookworm** | Oval, thin clear shell | Non-bile stained (Segmented inside) |
| **Trichuris** | Barrel-shaped with Bipolar plugs | "Double-ended plug" |
| **Enterobius** | Planoconvex (D-shaped) | "Flat on one side" |
| **T. solium** | Round with radial striations | "Wheel-like" appearance |
| **HNB (H. nana)** | Oval with polar filaments | "Twin shells" |

## 3. Fungal Identification Clues

| Feature | Target Organism | Visual Clue |
| :--- | :--- | :--- |
| **Germ Tube** | *Candida albicans* | Sprout-like outgrowth (No constriction) |
| **India Ink Halo** | *Cryptococcus* | Clear zone around cell (Capsule) |
| **Pilot Wheel** | *Paracoccidioides* | Multiple yeast budding (Mariner's wheel) |
| **Cigar Shaped** | *Sporothrix* | Yeast cells in tissue |
| **Acute Branch** | *Aspergillus* | 45-degree angle (V-shape) |
| **Obtuse Branch** | *Mucor* | 90-degree angle (T-shape) |

## 4. Viral Inclusions Identification
- **Owl-Eye**: CMV (Intranuclear inclusion with clear halo).
- **Negri Bodies**: Rabies (Intracytoplasmic, eosinophilic, round/oval).
- **Guarnieri Bodies**: Variola (Smallpox - intracytoplasmic).
- **Henderson-Patterson**: Molluscum contagiosum (Large, egg-shaped cytoplasmic).
