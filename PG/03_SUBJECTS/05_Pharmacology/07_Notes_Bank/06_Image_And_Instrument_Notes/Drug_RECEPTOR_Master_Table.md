# Drug Receptor Master Table: G-Protein Coupling

Mastering receptor signaling is essential for solving mechanisms of action questions in NEET PG.

## 1. The G-Protein Signaling Logic

| G-Protein | Primary Second Messenger | Key Effect |
| :--- | :--- | :--- |
| **Gs** (Stimulatory) | ↑ Adenylate Cyclase -> ↑ cAMP | Activation of PKA |
| **Gi** (Inhibitory) | ↓ Adenylate Cyclase -> ↓ cAMP | Inhibition of PKA |
| **Gq** (Phospholipase) | ↑ PLC -> ↑ IP3 / DAG / Ca2+ | Activation of PKC |

## 2. Receptor - G-Protein Mapping

| Receptor Type | G-Protein | Key Biological Effect |
| :--- | :--- | :--- |
| **Alpha-1** | **Gq** | Vascular smooth muscle contraction (Vasoconstriction) |
| **Alpha-2** | **Gi** | Inhibition of Norepinephrine release (Pre-synaptic) |
| **Beta-1** | **Gs** | ↑ Heart rate and contractility |
| **Beta-2** | **Gs** | Bronchodilation; Vasodilation in skeletal muscle |
| **Beta-3** | **Gs** | Thermogenesis; Detrusor muscle relaxation |
| **M1** | **Gq** | CNS excitation; Gastric acid secretion |
| **M2** | **Gi** | ↓ Heart rate (Vagal effect) |
| **M3** | **Gq** | Miosis; Bronchoconstriction; GIT motility; Glandular secretion |
| **D1** | **Gs** | Renal vasodilation |
| **D2** | **Gi** | Inhibition of prolactin; CNS signaling |
| **H1** | **Gq** | Allergic response; Bronchoconstriction |
| **H2** | **Gs** | ↑ Gastric acid secretion |
| **V1** (Vasopressin) | **Gq** | Vasoconstriction |
| **V2** (Vasopressin) | **Gs** | Water reabsorption in Renal Collecting Duct |

## 3. High-Yield "Switch" Mnemonic
- **Gs**: "**B**e **V**ery **H**appy" -> **B**eta receptors, **V**2, **H**2.
- **Gi**: "**M-A-D**" -> **M**2, **A**lpha-2, **D**2.
- **Gq**: "**H-A-V-1 M-1-M-3**" -> **H**1, **A**lpha-1, **V**1, **M**1, **M**3.
