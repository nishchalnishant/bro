# CVS, Endocrine, Respiratory, And GIT Drugs

## CVS
- Antihypertensives, antianginals, antiarrhythmics, heart failure drugs, anticoagulants, and antiplatelets.

## Endocrine
- Insulin, oral hypoglycemics, thyroid drugs, steroids, contraceptives, and infertility-related drugs.

## Respiratory And GIT
- Asthma drugs, antihistamines, cough drugs, antiulcer therapy, antiemetics, laxatives.

## Must Learn
- Antiarrhythmic classification.
- Antidiabetic mechanisms.
- Steroid adverse effects.
- Asthma stepwise therapy.
