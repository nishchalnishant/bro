# Revision System

Recommended revision logic:

- `01_Revision_1`: first revision after initial read
- `02_Revision_2`: second pass with active recall
- `03_Revision_3`: compressed high-yield pass
- `04_Last_45_Days`: test-heavy revision phase
- `05_Last_15_Days`: volatile topics only
- `06_Last_7_Days`: final rapid review

What to revise repeatedly:

- PYQ-heavy facts
- Volatile one-liners
- Images and instruments
- Scores, criteria, classifications
- Drug of choice and investigation of choice
