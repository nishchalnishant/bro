# NEET PG Preparation Workspace

This folder is structured to help you prepare end-to-end for the NEET PG exam.

Use it in this order:

1. Start in `00_START_HERE`.
2. Build your plan in `01_MASTER_PLAN`.
3. Store all study material in `02_RESOURCES`.
4. Study each subject inside `03_SUBJECTS`.
5. Capture revision material in `04_NOTES_BANK`.
6. Practice questions in `05_QUESTION_PRACTICE`.
7. Track tests and improve using `06_TESTS_AND_ANALYSIS`.
8. Run structured revisions in `07_REVISION_SYSTEM`.
9. Fix weaknesses using `08_WEAK_AREAS`.
10. Consolidate scoring topics in `09_HIGH_YIELD_AREAS`.
11. Define exam-day execution in `10_EXAM_STRATEGY`.
12. Protect consistency using `11_HEALTH_AND_DISCIPLINE`.
13. Store admin and counselling info in `12_ADMIN`.

Core rule:

- Every study session should produce one of these outputs: notes, solved questions, revision points, or mistakes logged.

Recommended workflow:

- Learn from a source.
- Make concise notes.
- Solve QBank/PYQs.
- Log mistakes.
- Revise the same topic in spaced cycles.
- Re-test weak topics.

Important:

- Keep only high-yield material in this workspace.
- Avoid storing duplicate notes from multiple sources.
- Prefer short active-recall notes over long passive notes.
